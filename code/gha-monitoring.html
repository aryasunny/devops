üöÄ GitHub Actions Workflow for Monitoring Deployment

This workflow builds and deploys monitoring infrastructure defined in AWS CDK ‚Äî CloudWatch alarms, dashboards, and SNS topics ‚Äî ensuring version-controlled and automated rollouts.

üß© .github/workflows/deploy-monitoring.yml


name: Deploy Monitoring (CDK)

on:
  push:
    branches: [ main ]
  workflow_dispatch:

permissions:
  id-token: write
  contents: read

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Configure AWS Credentials
        uses: aws-actions/configure-aws-credentials@v4
        with:
          role-to-assume: arn:aws:iam::123456789012:role/monitoring-deploy-role
          aws-region: ap-southeast-2

      - name: Install dependencies
        run: npm ci

      - name: Deploy monitoring stack
        run: npx cdk deploy monitoring-stack --require-approval never
